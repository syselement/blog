## Windows Host Machine

- The host PC uses the GPU for decrypting the passwords, so it's faster than using the CPU.
- NVIDIA GPUs require "[NVIDIA Driver](https://www.nvidia.com/download/index.aspx)" (440.64 or later) and "[CUDA Toolkit](https://developer.nvidia.com/cuda-downloads)" (9.0 or later)

- **Nvidia Drivers**

```bash
sudo apt update && sudo apt full-upgrade -y
sudo apt install linux-headers-$(uname -r) -y
sudo reboot

# Enable OpenCL for NVIDIA

# Check conflinting kernel module nouveau
lsmod | grep -i nouveau
echo -e "blacklist nouveau\noptions nouveau modeset=0\nalias nouveau off" | sudo tee /etc/modprobe.d/blacklist-nouveau.conf
sudo update-initramfs -u && sudo reboot
lsmod | grep -i nouveau

# Install OpenCL ICD loader, drivers and CUDA Toolkit
sudo apt install -y ocl-icd-libopencl1 nvidia-driver nvidia-cuda-toolkit nvidia-detect

sudo reboot

# Check drivers
nvidia-smi
nvidia-smi -i 0 -q
sudo nvidia-xconfig --query-gpu-info | grep 'BusID : ' | cut -d ' ' -f6
nvidia-detect
lspci | grep -i vga
lspci -s 01:00.0 -v
glxinfo | grep -i "direct rendering"
```
